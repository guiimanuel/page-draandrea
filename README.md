# Dra. Andrea Doria Astro

Landing page institucional criada em Astro para a Dra. Andrea Doria.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Estrutura

- `src/pages/index.astro`: pagina principal e dados editaveis da landing page.
- `src/styles/global.css`: estilos globais e responsivos.
- `public/assets/images`: imagens usadas pela pagina.
